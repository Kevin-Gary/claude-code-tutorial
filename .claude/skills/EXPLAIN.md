# What a skill is

A skill is a reusable recipe. Instead of re-explaining the same thing to Claude
every time, you capture it once as a SKILL.md file, and Claude reaches for it on
its own when it's relevant.

- This folder holds them. `brand-voice/SKILL.md` is an example: our marketing
  tone captured once, so every piece of copy comes out on-voice.
- You build them AS YOU GO. Did some work you'll repeat? Say "encode that into a
  skill" and Claude writes the file. Next time it's one sentence.
- Recipe, not a script. A skill is guidance Claude applies with judgment, not a
  fixed list of steps. (A fixed list of steps with a distinct output is a
  command, which lives in `.claude/commands/`.)
