# MCP - connect Claude to everything

MCP (Model Context Protocol) is an open standard Anthropic created in 2024 for
connecting AI to your tools and data.

- Plug Claude Code into Stripe, Postgres, Notion, Gmail, Calendar, HubSpot. About
  98% of the systems you use have an MCP now.
- For coding: it knows your real data shapes, so it writes better code.
- For running a business: ask one question ("which accounts are about to
  churn?") and it pulls and synthesizes across all of them at once.
- A "plugin" is just a bundle of skills + commands + an MCP. Nothing magic.
